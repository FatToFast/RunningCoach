# Knowledge Base Documents

이 디렉토리에 러닝 가이드/교재 마크다운 파일을 추가하세요.

## 파일 형식

- 마크다운 (.md) 파일만 지원됩니다
- `_`로 시작하는 파일과 이 README.md는 무시됩니다
- 파일명은 정렬 순서를 결정합니다 (예: `01_기초.md`, `02_훈련원칙.md`)

## 문서 작성 가이드

### 제목 구조

```markdown
# 문서 제목

## 섹션 1
내용...

## 섹션 2
내용...
```

- `#` 제목은 문서 전체 제목으로 사용됩니다
- `##` 제목은 청크 분할 기준이 됩니다 (각 섹션이 별도 청크로 저장)
- 각 섹션은 800자 기준으로 추가 분할될 수 있습니다

### 권장 사항

1. **명확한 섹션 제목**: AI가 관련 내용을 찾기 쉽도록 구체적인 제목 사용
2. **핵심 내용 우선**: 각 섹션의 시작 부분에 핵심 내용 배치
3. **적절한 길이**: 한 섹션당 500-1500자 권장

## 인덱스 빌드

문서 추가/수정 후 인덱스를 다시 빌드해야 합니다:

```bash
cd backend
GOOGLE_AI_API_KEY=your-key python scripts/build_knowledge_index.py
```

## 예시 파일

```markdown
# 마라톤 훈련 원칙

## 점진적 과부하 원칙
훈련 부하는 주당 10% 이내로 점진적으로 증가시켜야 합니다.
급격한 훈련량 증가는 부상의 주요 원인입니다...

## 회복의 중요성
훈련 효과는 회복 중에 나타납니다.
충분한 수면(7-9시간)과 영양 섭취가 필수적입니다...

## 주기화 (Periodization)
훈련은 기초기, 강화기, 시합기, 회복기로 나뉩니다...
```
