# RunningCoach 사용자 가이드

RunningCoach를 사용하여 더 나은 러너가 되세요!

---

## 시작하기

### 1. 계정 생성 및 로그인

1. 앱에 접속합니다
2. 이메일과 비밀번호로 로그인합니다
3. 처음 사용하는 경우 관리자에게 계정 생성을 요청하세요

### 2. Garmin Connect 연동

1. **설정** 페이지로 이동합니다
2. **Garmin 연동** 섹션에서 Garmin 계정 정보를 입력합니다
3. **연결** 버튼을 클릭합니다
4. 연결이 완료되면 **동기화** 버튼으로 데이터를 가져옵니다

> 첫 동기화 시 모든 활동을 가져오므로 시간이 걸릴 수 있습니다.

---

## 주요 기능

### 대시보드

대시보드에서 러닝 현황을 한눈에 확인할 수 있습니다:

- **주간 주행거리**: 이번 주 달린 거리
- **피트니스 지표**: CTL(체력), ATL(피로), TSB(컨디션)
- **최근 활동**: 최근 러닝 기록
- **훈련 페이스**: VDOT 기반 권장 페이스

#### 피트니스 지표 이해하기

| 지표 | 의미 | 좋은 상태 |
|------|------|----------|
| CTL (Chronic Training Load) | 장기 체력 수준 | 꾸준히 증가 |
| ATL (Acute Training Load) | 최근 피로도 | 너무 높지 않게 |
| TSB (Training Stress Balance) | 현재 컨디션 | +10 ~ -10 |

### 활동 목록

모든 러닝 활동을 확인할 수 있습니다:

1. **활동** 메뉴 클릭
2. 날짜별로 정렬된 활동 목록 확인
3. 활동 클릭하여 상세 정보 보기

#### 활동 상세 페이지

- **지도**: GPS 경로 시각화
- **랩 데이터**: 킬로미터별 페이스, 심박
- **심박 존**: 각 존에서 보낸 시간
- **페이스 차트**: 거리별 페이스 변화

### AI 코치

AI 코치와 대화하며 맞춤형 훈련 조언을 받으세요:

1. **AI 코치** 메뉴 클릭
2. 질문이나 요청을 입력합니다

#### 사용 예시

```
"다음 마라톤 준비를 위한 12주 훈련 계획을 세워줘"
"이번 주 훈련이 너무 힘들었어, 다음 주는 어떻게 해야 할까?"
"내 VDOT에 맞는 인터벌 훈련을 추천해줘"
"왜 최근 페이스가 느려졌을까?"
```

#### AI 코치가 참조하는 정보

- 최근 활동 데이터
- 피트니스 지표 (CTL/ATL/TSB)
- 대회 목표 (설정한 경우)
- VDOT 및 훈련 페이스

### 워크아웃

맞춤형 워크아웃을 생성하고 Garmin에 전송하세요:

1. **워크아웃** 메뉴 클릭
2. **새 워크아웃 만들기** 클릭
3. 워크아웃 이름과 단계를 입력합니다
4. **Garmin으로 전송** 클릭

#### 워크아웃 예시

**인터벌 훈련**:
1. 워밍업: 10분 쉬운 페이스
2. 반복 5회: 1km 인터벌 + 2분 휴식
3. 쿨다운: 10분 쉬운 페이스

### 트렌드

장기적인 훈련 추이를 분석합니다:

- **주간 주행거리 변화**
- **평균 페이스 추이**
- **심박수 변화**
- **훈련 부하 그래프**

### 캘린더

월간 훈련 현황을 캘린더 형태로 확인합니다:

- 각 날짜의 훈련 내용
- 계획된 워크아웃
- 휴식일 확인

### 개인 기록

거리별 최고 기록을 추적합니다:

- 1km, 5km, 10km, 하프마라톤, 마라톤
- 기록 달성 날짜
- 페이스 및 심박 정보

### 장비 관리

러닝화 수명을 관리하세요:

1. **장비** 메뉴 클릭
2. **신발 추가** 클릭
3. 신발 정보 입력 (이름, 브랜드, 구매일)
4. 활동과 연결하여 거리 추적

> 일반적으로 러닝화는 500-800km 사용 후 교체를 권장합니다.

## Strava 연동

활동을 Strava에 자동으로 업로드할 수 있습니다:

### 설정 방법

1. **설정** → **Strava 연동**
2. **Strava 연결** 클릭
3. Strava 로그인 및 권한 승인
4. **자동 업로드** 옵션 활성화 (선택)

### 수동 업로드

자동 업로드를 사용하지 않는 경우:

1. 활동 상세 페이지로 이동
2. **Strava로 내보내기** 버튼 클릭

---

## 대회 목표 설정

목표 대회를 설정하면 AI 코치가 더 맞춤화된 조언을 제공합니다:

1. **설정** → **대회 목표**
2. **새 목표 추가** 클릭
3. 정보 입력:
   - 대회명
   - 대회 날짜
   - 거리 (5K, 10K, 하프, 풀)
   - 목표 기록 (선택)

## 훈련 페이스 이해하기

VDOT 기반 훈련 페이스 존:

| 존 | 목적 | 체감 강도 |
|---|------|---------|
| **Easy (E)** | 기초 체력, 회복 | 대화 가능 |
| **Marathon (M)** | 마라톤 페이스 적응 | 편안하지만 집중 필요 |
| **Threshold (T)** | 젖산 역치 향상 | 힘들지만 유지 가능 |
| **Interval (I)** | VO2max 향상 | 매우 힘듦 |
| **Repetition (R)** | 스피드, 폼 개선 | 최대 노력 |

### 권장 훈련 비율

- **Easy**: 전체 훈련의 70-80%
- **Quality (M/T/I/R)**: 20-30%

## 자주 묻는 질문

### Q: 동기화가 안 돼요

**A**: 다음을 확인해주세요:
1. Garmin 계정 정보가 올바른지
2. 인터넷 연결 상태
3. 설정 페이지에서 다시 연결 시도

### Q: VDOT은 어떻게 계산되나요?

**A**: 최근 레이스 기록 또는 타임 트라이얼 결과를 기반으로 Jack Daniels 공식을 사용하여 계산됩니다.

### Q: AI 코치 응답이 느려요

**A**: AI 응답은 복잡한 질문일수록 시간이 걸릴 수 있습니다. 잠시 기다려주세요.

### Q: 이전 활동이 안 보여요

**A**: 설정에서 **전체 동기화**를 실행하면 과거 모든 활동을 가져옵니다.

### Q: 심박 존이 이상해요

**A**: 설정에서 최대 심박수와 안정시 심박수를 정확히 입력해주세요.

## 팁 & 트릭

### 효과적인 AI 코치 활용

1. **구체적으로 질문하세요**
   - "훈련 추천해줘" → "5km 기록 향상을 위한 이번 주 훈련 추천해줘"

2. **목표를 설정하세요**
   - 대회 목표가 설정되면 AI가 더 맞춤화된 조언 제공

3. **정기적으로 대화하세요**
   - 주간 피드백을 받으면 훈련 조정이 쉬워짐

### 부상 예방

1. 주간 주행거리 증가는 10% 이내로
2. 쉬운 러닝 비율 70% 이상 유지
3. TSB가 -20 이하면 휴식 필요
4. 러닝화 교체 시기 확인

## 문제 해결

문제가 발생하면:

1. 브라우저 새로고침
2. 로그아웃 후 다시 로그인
3. 다른 브라우저에서 시도
4. 관리자에게 문의
